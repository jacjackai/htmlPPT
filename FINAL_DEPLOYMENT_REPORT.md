## 🚀 快速开始

### 克隆项目
```bash
git clone https://github.com/jacjackai/htmlPPT.git
cd htmlPPT
```

### 直接使用
```bash
# 打开主应用
open src/index.html

# 或者使用Python启动本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000/src/index.html
```

### 使用示例
```bash
# 查看基础示例
open examples/basic.html

# 查看演示页面
open examples/demo.html
```

## 🎨 主要功能

1. **幻灯片管理** - 创建、编辑、删除、排序幻灯片
2. **演示模式** - 全屏演示，支持键盘控制
3. **主题系统** - 8种预设主题（浅色、深色、商务、创意等）
4. **模板系统** - 10种专业模板（标题、内容、图片、图表等）
5. **自动保存** - 本地存储，永不丢失
6. **历史记录** - 支持撤销/重做
7. **快捷键** - 30+快捷键，提升效率
8. **协作功能** - 评论和批注系统
9. **导出功能** - 支持JSON、HTML、Markdown、PDF等格式
10. **工具函数** - 丰富的工具函数库
