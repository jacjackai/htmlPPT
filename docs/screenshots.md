## 截图说明

这些截图展示了HTML PPT的主要功能和界面。

### 如何添加截图

1. 在`assets/images/`目录下添加截图文件
2. 使用以下命令生成截图：
   ```bash
   # macOS
   screencapture -x assets/images/main-interface.png

   # Linux
   import -window root assets/images/main-interface.png

   # Windows
   # 使用截图工具保存到assets/images/目录
   ```

3. 更新此文件中的图片路径

### 截图建议

- 使用高分辨率（至少1920x1080）
- 展示关键功能
- 保持界面整洁
- 使用深色和浅色主题各一张
- 包含移动端截图

### 在线工具

可以使用以下在线工具生成截图：

- [Screenshot API](https://screenshotapi.net/)
- [Urlbox](https://urlbox.io/)
- [BrowserStack](https://www.browserstack.com/screenshots)
